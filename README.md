# PB

Projekt w ramach modelowania i analizy systemów informatycznych. 

Zaprojektowanie procesu przyjmowania i oceniania zamówień dla firmy poligraficznej. 

Wykorzystanie Activiti jako silnika procesu BPMN. Spring boot wykorzystany między innymi do połączenia się się serwerem mailowym,
aby realizować taski mailowe.
